### M3.1 遗忘曲线引擎 🔲 待实现

- **文件**: `src/algomate/core/memory/forgotten_curve.py`
- **功能描述**: 基于艾宾浩斯遗忘曲线计算复习时间节点
- **输入**:
  - `card_id`: 卡牌ID
  - `card_created_at`: 卡牌创建时间
  - `card_last_reviewed`: 上次复习时间
- **输出**:
  - `next_review_date`: 下次复习日期
  - `review_level`: 复习等级（1-7）
  - `is_due`: 是否应该复习
- **核心算法**:

```
复习节点：创建 → 1天 → 3天 → 7天 → 14天 → 30天 → 60天

每次复习后，下一个复习间隔翻倍：
间隔倍数：[1, 2, 4, 2, 2, 2] 对应 [1天, 3天, 7天, 14天, 30天, 60天]

复习成功后推进到下一个节点
复习失败后退回到前一个节点
```

### M3.2 耐久度管理系统 🔲 待实现

- **文件**: `src/algomate/core/game/durability.py`
- **功能描述**: 管理卡牌耐久度的增减
- **输入**: 卡牌ID、操作类型（复习成功/失败/未复习）
- **输出**: 更新后的耐久度值
- **依赖**: M2.3 卡牌模型

**耐久度变化规则**:

| 操作                | 条件     | 耐久度变化 | 难度系数                     |
| ------------------- | -------- | ---------- | ---------------------------- |
| 复习成功/Boss击败   | -        | +20        | 简单×0.5，普通×1.0，困难×1.5 |
| 复习失败/Boss未击败 | -        | -5         | 简单×0.5，普通×1.0，困难×1.5 |
| 每日未复习          | 每天衰减 | -2         | 简单×0.5，普通×1.0，困难×1.5 |

**公式**:
```
复习成功: durability = min(100, durability + 20 * difficulty_multiplier)
复习失败: durability = max(0, durability - 5 * difficulty_multiplier)
每日衰减: durability = max(0, durability - 2 * difficulty_multiplier)
濒危阈值: durability < 30
消散阈值: durability = 0 → is_sealed = True
```
